# OPN Staking DeFi

A secure and efficient decentralized staking platform built for the OPN Chain ecosystem. Submitted for **OPN Builders Season 1**.

## Overview
OPN Staking allows users to stake their tokens and earn passive rewards. The platform is designed with a focus on security, simplicity, and leveraging the high-speed infrastructure of OPN Chain.

## Features
- **Secure Staking:** Integrated with OpenZeppelin's `ReentrancyGuard` to prevent common attack vectors.
- **Dynamic Rewards:** Rewards are calculated accurately based on the time staked (per second) and the user's balance.
- **EVM Compatible:** 100% ready to be deployed on the OPN Chain Testnet and Mainnet.

## Tech Stack
- **Smart Contracts:** Solidity ^0.8.20
- **Libraries:** OpenZeppelin Contracts
- **Network:** OPN Chain (Testnet)

## Repository Structure
- `OPNStaking.sol` : The core logic for staking, withdrawing, and claiming rewards.
